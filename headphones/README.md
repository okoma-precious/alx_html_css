# 🎧 Headphones Landing Page

This project is a responsive landing page for a fictional headphones brand. The page was built using only **HTML and CSS**, based on a design provided in a **Figma file**. It demonstrates best practices in layout structuring, responsiveness, and clean user interactions.

## 🚀 Project Features

- Fully responsive layout (desktop and mobile)
- Mobile-first design: layout adjusts at 480px width or less
- Styled using custom fonts: `Source Sans Pro` and `Spin Cycle OT`
- Button and link interaction effects (hover/active)
- Clean and semantic HTML5 structure
- Max content width: 1000px centered on the page
